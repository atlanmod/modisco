= Atlanmod Modisco

Atlanmod Modisco is a fork of https://www.eclipse.org/MoDisco/[Eclipse MoDisco] project.


## Local build

Building the project:

[source,shell script]
----
git clone git@github.com:atlanmod/modisco.git
cd modisco
mvn clean install
----

Building the Eclipse plugins:

[source,shell script]
----
cd plugins/eclipse
mvn clean install
----
